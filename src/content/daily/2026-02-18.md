## 科研

- [x] Proactive 系统在 ESTP-Bench 上测试，尝试 5 种优化假设
  - 有效方案：Logprob gap = logP(Yes) - logP(No)，三段规则 + 12s Cooldown
  - 无效方案：修改 prompt（prompt level 无法干预 yes-bias）、自适应时间窗口、置信度校准
- [x] 确认 yes-bias 是 VLM 在主动触发任务上的系统性倾向，非单个模型问题
- [x] 对 Adaptive-D 进行超参数调优，验证最佳方案
  - D 有效：Info Fn(+0.075)、Text-Rich(+0.069)、Task Understanding(+0.057)
  - D 有害：Action Recognition(-0.146)、Attribute Perception(-0.123)
- [x] 尝试训练 MLP 触发器（特征：goal_state、step_progress、stagnation 等）
- [ ] 下一步：把"状态预测"单独落盘 → 做两阶段决策 → 升级到轻量策略网络
- [ ] 分离两类目标：info-assist 和 safety-corrective
